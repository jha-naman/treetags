Downloadable language plugins for [TreeTags](https://github.com/jha-naman/treetags), grouped by the plugin ABI version your treetags build supports. The CLI selects the right set automatically:

```
treetags plugin available     # list plugins your build can install
treetags plugin install NAME  # install one
```
